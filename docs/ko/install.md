# 설치

## 요구 사항

- Unity (VRChat 권장 버전)
- VRChat Creator Companion (VCC)
- VRChat SDK3 Avatar
- NDMF **1.10+** 또는 Modular Avatar **1.15+**

---

## 설치 방법 (VCC)

<div class="psha-vcc-actions" markdown>
<a class="md-button md-button--primary psha-vcc-add"
   href="vcc://vpm/addRepo?url=https://psha1220.github.io/Modular-Emote/index.json">
  Add to VCC
</a>

<a class="md-button psha-icon-btn"
   href="https://psha1220.github.io/Modular-Emote/"
   target="_blank" rel="noopener noreferrer"
   title="Listing / 안내 페이지">
  i
</a>
</div>

1. 위의 **Add to VCC** 버튼을 눌러 VCC에 Listing을 추가합니다.  
2. VCC → **Projects**에서 대상 프로젝트의 **Manage Project** 클릭  
3. **Psha Modular Emote**를 찾아 최신 버전으로 **Add Package(+)** 합니다.
4. Unity프로젝트를 실행
5. 컴파일이 끝날 때까지 기다립니다.
